# 📊 Embedded C Statistics Project

## 👤 Author
Samaresh Mozumder

## 📌 Description
This project analyzes an array of unsigned char data and computes:

- Maximum value
- Minimum value
- Mean (rounded down)
- Median (rounded down)

The array is also sorted in descending order.

## 📂 Files
- stats.c → Implementation
- stats.h → Header file
- README.md → Documentation

## ▶️ Compilation & Execution

```bash
gcc stats.c -o stats
./stats
